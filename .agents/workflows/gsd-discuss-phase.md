# /gsd:discuss-phase [N]

Run this BEFORE `/gsd:plan-phase` for any phase. This is where Skmoonly's decisions get captured.
**Never skip this step.**

## Steps

1. Read `ROADMAP.md` for the target phase tasks.
2. Read `STATE.md` for current project status and any existing decisions.
3. Read `REQUIREMENTS.md` for any open questions relevant to this phase.
4. Present to Skmoonly:
   - A summary of what Phase N covers
   - All open questions from REQUIREMENTS.md that affect this phase
   - Any design or tech decisions that need to be made before work starts
   - Options when a decision could go multiple ways
5. Wait for Skmoonly's responses.
6. Record all decisions in `STATE.md` under "Active Decisions".
7. Update `STATE.md` with `Active Phase: N — Discussed, ready for /gsd:plan-phase N`.
8. Confirm with Skmoonly that all blockers are resolved before proceeding.
